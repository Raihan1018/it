# Modern QA & Software Testing Landing Page

A high-performance, aesthetically pleasing contact and landing page built with the React and Tailwind CSS**. This project features a fully functional contact system that transmits data directly to your email without a dedicated backend.

🚀 **Live Demo:** [https://it-gray-zeta.vercel.app/](https://it-gray-zeta.vercel.app/)

![Project Preview](https://via.placeholder.com/800x400?text=QA+Testing+Landing+Page+Preview)

## 🚀 Key Features


- **Modern UI/UX:** Built with **Tailwind CSS** and **DaisyUI** for a clean, professional aesthetic.
- **High-End Animations:** Utilizing **Framer Motion** for smooth scroll reveals and interactive button states.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
- **Dynamic Maps:** Integrated Google Maps with a custom grayscale filter and hover interaction.
- **Developer Focused:** Includes a "Developed by" section linked to GitHub profiles.

## 🛠️ Tech Stack

- **Frontend:** React.js 
- **Styling:** Tailwind CSS, Framer Motion (Animations), React Icons
- **Icons:** Hi2 (Heroicons), Fi (Feather Icons), Fa (Font Awesome)

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Raihan1018/it.git
   cd it
   ```
2. Install dependencies:

```Bash
npm install
```

3. Start the development server:

```Bash
npm run dev
```
